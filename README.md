# 🛒 Mugen Stock App

Une application de gestion de stock simple et efficace pour tracker tes produits en temps réel.

## 📋 Fonctionnalités

✅ **Ajouter des produits** - Désignation, Référence, Quantité  
✅ **Voir ton stock** - Liste complète ou par référence  
✅ **Modifier les quantités** - Ajouter ou retirer des pièces  
✅ **Supprimer des produits** - Gestion simple  
✅ **Persistance des données** - Sauvegarde en JSON  

## 🚀 Phases de développement

### Phase 1 ✅ TERMINÉE
- Console app en Python pur
- Fichier JSON pour le stockage
- Menu interactif

### Phase 2 (À venir)

## 🛠️ Installation

**Prérequis :**
- Python 3.8+

**Étapes :**
1. Clone ce repo
2. Va dans le dossier : `cd mugen-stock-app`
3. Lance l'app : `python stock_app.py`

## 📖 Utilisation
1️⃣  Ajouter un produit
2️⃣  Voir le stock
3️⃣  Ajouter des pièces
4️⃣  Retirer des pièces
5️⃣  Supprimer un produit
6️⃣  Chercher par référence
7️⃣  Quitter
